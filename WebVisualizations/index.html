Summary: Latitude vs X

This assignment collected weather data for over 500 cities from various latitudes around the world. The data was collected using python scripts and API's. Various python libraries were used including matplotlib, pandas, numpy, requests, and more. Data was collected using the OpenWeatherMap API.

Weather data was collected for each city including temperature, humidity, cloudiness, and wind speed. The values were then plotted against each cities' latitude and then analized for trends in the data. Click on the "plots" pages for an in-depth analysis of each plot.
